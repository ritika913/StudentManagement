About This Project

The Student Management System is a web-based application designed to simplify the process of managing student records. It allows users to add, view, update, and delete student information easily through a user-friendly interface.

Built using HTML, CSS, JavaScript, and powered by a Spring Boot + MySQL backend, this system is a complete full-stack solution tailored for academic institutions or administrative users.

Features include:

  • Secure user registration and login
  • Responsive design for desktop and mobile
  • Real-time student data management
  • Clean UI with sidebar navigation

Developed in 2025 by Ritika Rajak. For suggestions or improvements, feel free to contribute or reach out.
